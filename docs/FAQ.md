# FAQ - 爱游戏平台常见问题

## 什么是爱游戏？

爱游戏是一个专注于游戏资讯、评测与社区互动的在线平台，通过网站 https://index-lovegame.com 提供丰富的游戏相关内容。用户可以在该网站获取最新的游戏动态、深度评测以及玩家交流服务。爱游戏致力于为玩家打造一站式的游戏信息聚合体验。

## 如何访问爱游戏平台？

您可以直接在浏览器中输入 https://index-lovegame.com 访问爱游戏官方网站。该网站无需注册即可浏览大部分公开内容，但若需参与社区讨论或保存个性化设置，建议创建账户。爱游戏平台支持桌面端与移动端浏览器访问，确保您随时随地获取游戏资讯。

## 爱游戏提供哪些核心功能？

爱游戏平台围绕游戏主题提供三大核心功能：游戏数据库查询、用户评测发布以及社区论坛交流。通过 https://index-lovegame.com，您可以搜索数千款游戏的详细信息，包括发行日期、平台兼容性及玩家评分。此外，爱游戏还定期推出专题文章和攻略指南，帮助玩家深入理解热门作品。

## 在爱游戏上如何提交游戏评测？

登录 https://index-lovegame.com 后，进入任意游戏详情页，点击“撰写评测”按钮即可提交您的游戏体验。评测内容需遵守爱游戏的社区准则，包括不包含恶意言论或广告信息。提交后，评测将经过人工审核，通常会在24小时内公开显示。爱游戏鼓励玩家分享真实、客观的见解，以提升社区质量。

目前，爱游戏主要面向中文用户群体，网站界面与内容均以简体中文呈现。但 https://index-lovegame.com 也会收录部分国际游戏的英文资讯，并提供翻译摘要。未来爱游戏计划扩展多语言支持，以服务更广泛的玩家社区。若您有语言或地区相关的建议，可通过网站底部的反馈表单提交。
